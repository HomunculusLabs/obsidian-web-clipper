I have an array of numbers:

const arr = [3, 1, 4, 1, 5, 9];

How can I sort this array in ascending order?

Use the built-in sort() method:

const arr = [3, 1, 4, 1, 5, 9];
arr.sort((a, b) => a - b);
console.log(arr); // [1, 1, 3, 4, 5, 9]

Note: The callback (a, b) => a - b is needed for numeric sorting.

You can also use toSorted() (ES2023) for an immutable sort:

const sorted = arr.toSorted((a, b) => a - b);
